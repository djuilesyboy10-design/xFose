# xFOSE - Extended Fallout 3 Script Extender

**xFOSE is a modernized continuation of the original FOSE (Fallout 3 Script Extender), bringing new life to Fallout 3 modding after 15 years of stagnation.**

## What is xFOSE?

xFOSE is a drop-in replacement for the original FOSE 1.7. It maintains 100% backward compatibility with all existing FOSE plugins and mods, while introducing powerful new capabilities for mod developers.

## Key Features

- **Full Backward Compatibility** — All existing FOSE plugins and mods work without modification
- **Working Event Manager** — Script event handlers (OnHit, OnDeath, OnLoad, OnEquip, and more) now function correctly
- **Modern Codebase** — Updated and maintained for stability and future expansion
- **Open Source** — Free to use, modify, and improve by anyone

## For Players

Simply install xFOSE instead of the original FOSE. Your existing mods will work exactly as before, with the added benefit of supporting newer mods that use xFOSE's enhanced features.

## For Modders

xFOSE unlocks the full potential of FOSE's Event Manager API, allowing you to:
- Respond to game events in real-time (hits, deaths, equips, loads, etc.)
- Create dynamic gameplay mechanics
- Build more responsive and interactive mods

See the included Developer README for detailed API documentation and examples.

## Installation

1. Download the xFOSE package for your Fallout 3 version
2. Extract `xfose_loader.exe` and `xfose_1_7.dll` to your Fallout 3 game folder (next to Fallout3.exe)
3. Launch the game using `xfose_loader.exe`
4. Existing FOSE plugins will load automatically from `Data\FOSE\Plugins\`

## Version

This is xFOSE 1.0, based on FOSE 1.7.0.3 with the Event Manager fully restored and modernized.
