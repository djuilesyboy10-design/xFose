xFOSE Changelog
===============

Version 1.0 - Initial Release
============================

DataInterface Enhancements
--------------------------
- Added kFOSEData_NumLoadedPlugins data ID (2) - Returns number of loaded plugins
- Added kFOSEData_FOSEVersion data ID (3) - Returns FOSE version number (0x01030032 for FOSE 1.3.2)
- Added DataInterface_GetData function to expose runtime data values
- Added ClearScriptDataCache function for performance optimization

FormExtraData System (NEW)
---------------------------
- Implemented complete FormExtraData manager for managing extra data attached to game forms
- Added kFOSEData_FormExtraDataGet (1) - Get extra data by name from a form
- Added kFOSEData_FormExtraDataGetAll (2) - Get all extra data from a form
- Added kFOSEData_FormExtraDataAdd (3) - Add extra data to a form
- Added kFOSEData_FormExtraDataRemoveByName (4) - Remove extra data by name
- Added kFOSEData_FormExtraDataRemoveByPtr (5) - Remove extra data by pointer
- Added kFOSEData_FormExtraDataCount (6) - Count extra data entries on a form
- Added kFOSEData_FormExtraDataExists (7) - Check if extra data exists on a form
- Added kFOSEData_FormExtraDataIterate (8) - Iterate through all extra data on a form
- All FormExtraData functions use thread-safe critical section access
- Created FormExtraData.cpp and FormExtraData.h files

ArrayVarManager Enhancements
----------------------------
Simple Enhancements:
- Added kFOSEData_ArrayVarMapDeleteBySelf (9) - Delete array by self (cleanup function)
- Added kFOSEData_ArrayVarInsertAt (11) - Insert element at specific index in array
- Added kFOSEData_ArrayVarFindByValue (12) - Find element by value, returns index
- Added kFOSEData_ArrayVarCountByType (13) - Count elements by type (Integer, Float, String)

Medium Enhancements:
- Added kFOSEData_ArrayVarSort (18) - Sort array elements in ascending order
  * Elements sorted first by type, then by value
  * Uses std::sort with custom comparator
- Added kFOSEData_ArrayVarReverse (19) - Reverse array element order
  * Uses std::reverse
- Added kFOSEData_ArrayVarShuffle (20) - Randomly shuffle array elements
  * Uses std::random_shuffle
- Added #include <algorithm> to ArrayVar.cpp for sort/reverse/shuffle functions

StringVarManager Enhancements
-----------------------------
Simple Enhancements:
- Added kFOSEData_StringVarMapDeleteBySelf (10) - Delete string by self (cleanup function)
- Added kFOSEData_StringVarGetLength (14) - Get string length
- Added kFOSEData_StringVarCompare (15) - Compare two strings (returns 0 if equal)
- Added kFOSEData_StringVarConcatenate (16) - Concatenate two strings (creates new string)
- Added kFOSEData_StringVarSubstring (17) - Extract substring (creates new string)

Medium Enhancements:
- Added kFOSEData_StringVarUppercase (21) - Convert string to uppercase (in-place)
- Added kFOSEData_StringVarLowercase (22) - Convert string to lowercase (in-place)
- Added kFOSEData_StringVarTrim (23) - Remove leading/trailing whitespace (in-place)
  * Handles spaces, tabs, newlines, and carriage returns
- Added kFOSEData_StringVarReplace (24) - Replace occurrences of substring (in-place)
  * Returns number of replacements made

DataInterface Function Wiring
-----------------------------
- Updated DataInterface_GetFunc in DataInterface.cpp to return all new function pointers
- Added static wrapper functions for all non-static member functions
- All functions exposed through DataInterface with proper function IDs
- Total: 24 DataInterface functions now available to plugins
